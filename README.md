# TypeScript Test Repo with Jest
Welcome to the TypeScript Test Repository! This repository is set up for testing TypeScript code using Jest, a delightful JavaScript testing framework.

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/JohanWRutberg/TestingTS

2. Navigate to the project directory:
   cd TestinfTS

3. Install dependencies:
   npm install

Running Tests
4. Execute the following command to run the Jest tests:
   npm test

   This will run all the test suites and provide you with the test results.

Writing Tests
The test files are located in each projects directory. You can start writing your TypeScript tests using Jest's syntax. For more information on writing tests with Jest, refer to the official documentation.

Configuration
The Jest configuration can be found in the jest.config.js file. Feel free to customize it based on your project requirements.

Contributing
If you'd like to contribute, please fork the repository and create a pull request. Make sure to follow the existing coding style and provide appropriate test coverage.
